# Wordle Solver

A GitHub Pages website for solving Wordle puzzles.

## Setup

This project is set up to be hosted on GitHub Pages. The main page is `index.html`.

## Development

Simply open `index.html` in your browser to view the site locally.

## GitHub Pages

This site is automatically deployed to GitHub Pages when you push to the `main` branch.

Visit the live site at: https://thespudmore.github.io/wordle_solver/
